# A Journey to Srijita — Cinematic Birthday Site

This project is a small React + Tailwind + Framer Motion experience: "A Journey to Srijita" — an anime-inspired cinematic interactive birthday site.

Features
- Cinematic intro with starfield and glitch text
- JoJo-style dramatic build-up
- Emotional scroll journey with parallax
- Big reveal with confetti and heart particles
- Sequential love messages with typewriter effect
- Music toggle (3 styles), replay button, cursor particle trail

Local setup

1. Node.js 18+ recommended
2. Install dependencies:

```bash
npm install
```

3. Run development server:

```bash
npm run dev
```

Notes
- Icons use `lucide-react`.
- Audio is played using the WebAudio helper in `src/utils/audio.js`.
- For production build run `npm run build`.

Enjoy — press the replay button to watch the cinematic again.
